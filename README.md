# 칙칙퐁퐁

이 프로젝트는 실시간 멀티 Pong 게임을 위한 SPA 웹사이트입니다.

OAuth 사용하여 로그인하며, 2차 인증, 프로필 사진 업로드, 닉네임 변경, 채팅 등의 기능이 있습니다.

구글과 네이버를 통해 로그인/회원가입을 진행할 수 있습니다.

## 기술 스택
- Prettier
- Docker(Docker-compose)
### FrontEnd
- React
- TypeScript
- HTML / CSS / JavaScript

### BackEnd
- Nest.js
- PostgreSQL

## 실행
